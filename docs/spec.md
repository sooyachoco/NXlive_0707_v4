# NXlive — 넥슨 게임 라이브 스트리밍 플랫폼 (MVP)

## 개요

넥슨 게임 유저가 라이브 방송을 **발견하고 시청**하는 웹 스트리밍 플랫폼. Mock 데이터 기반 프론트엔드 MVP로, 실제 스트리밍/백엔드/인증 없이 발견→시청 핵심 UX를 검증한다.

- 대상: 넥슨 게임 외부 유저 (익명, **인증 없음**)
- 디자인: NX Basic 1.0v 토큰(라이트) — 좌측 사이드바 + 콘텐츠 (design-direction.md `★ 확정안`)
- 스택: React 19 + Vite + TS + Tailwind, 상태관리 Zustand, 라우팅 react-router-dom
- 데이터: `src/mocks/` 정적 Mock (VITE_USE_MOCK=true)

## 사용자 시나리오

- **S1 (발견→시청)**: 홈에서 실시간 인기 라이브를 보고, 카드를 눌러 라이브 시청 페이지로 진입해 방송을 본다.
- **S2 (탐색)**: 인기 스트리머 영역/편성표에서 관심 스트리머를 찾아 채널로 이동한다.
- **S3 (클립 소비)**: 클립 피드에서 하이라이트를 훑어본다.
- **S4 (검색)**: 게임/스트리머/클립을 통합 검색으로 찾는다.

## 기능 요구사항

### 페이지 목록 (5)

| 페이지 | 경로 | 대응 기능 |
|---|---|---|
| 홈 | `/` | F-01, F-02, F-06 |
| 라이브 시청 | `/watch/:streamId` | F-03 |
| 채널 | `/channel/:channelId` | F-04 |
| 클립 | `/clips` | F-05 |
| 검색 | `/search` | F-06 |

> 공통 레이아웃: 좌측 고정 사이드바(홈/게임/클립/채널/e스포츠/검색) + 상단 검색바.

### 기능별 요구사항

- **F-01 홈 발견 피드**: 대표 라이브(히어로) + 실시간 랭킹 리스트 + 실시간 라이브 카드 그리드(게임/제목/스트리머/시청자수/LIVE 배지).
- **F-02 인기 스트리머 + 편성표**: 홈 하단에 인기 스트리머 카드(아바타·게임·팔로워·팔로우 버튼) + 오늘의 편성표(스트리머 × 시간대 TV-guide).
- **F-03 라이브 시청**: 플레이어(Mock 썸네일/영상) + 방송 정보(제목·게임·스트리머·시청자수) + 실시간 채팅(Mock 메시지 자동 추가) + 추천 라이브 사이드.
- **F-04 채널**: 스트리머 프로필(배너·아바타·팔로워·소개) + 현재 방송 상태 + 지난 방송/클립 목록.
- **F-05 클립 피드**: 하이라이트 클립 카드 그리드(썸네일·제목·게임·조회수·스트리머), 게임 필터.
- **F-06 통합 검색**: 키워드로 게임/스트리머/클립 통합 검색, 탭 필터(전체/게임/스트리머/클립), 결과 없음 상태.

### 공통 UI 상태 (모든 목록/피드)

- 로딩 상태(스켈레톤), 빈 상태(결과 없음 안내), 기본 에러 문구.

## 상태 관리

- 감지 엔티티(6): Game, Channel(Streamer), LiveStream, Clip, ChatMessage, ScheduleProgram
- 자동 선택: **Zustand** (3~6개 구간). 서버 상태 없음(Mock) → React Query 불필요.

## 비기능 요구사항

- **성능**: 초기 로드 < 2s(로컬), 라우트 전환 즉시. 이미지/썸네일은 Mock 그라디언트.
- **반응형**: 데스크톱 우선(max-width 1440), ≤960px 사이드바 수평화·편성표 가로 스크롤.
- **접근성**: WCAG AA 대비, 키보드 포커스, aria 레이블(내비/검색/플레이어).
- **보안**: 인증/개인정보 없음. 외부 요청 없음.

## 제약사항

- 인증 없음(none), 백엔드 없음, 실제 스트리밍 없음 — 전부 Mock.
- 넥슨 GNB 미사용.
- NX Basic 토큰(라이트) 준수 — 임의 장식/변주 최소화.

## 용어 정의

- **라이브(LiveStream)**: 현재 방송 중인 스트림.
- **채널(Channel)**: 스트리머 단위 페이지/식별자.
- **클립(Clip)**: 방송 하이라이트 짧은 영상.
- **편성표(Schedule)**: 스트리머별 시간대 방송 계획.
