# 기능 흐름 (flow.md) — NXlive MVP

> ui-design-workflow STEP 1 — scenario.md §3 로드해 락 (재질문 없음, 흐름 소유는 write-scenario).

```
[진입] 홈(/)
  ├─ 발견(대표 라이브·실시간 그리드·인기 스트리머·편성표)
  │     └─(클릭)→ ★[시청] /watch/:id (플레이어+정보+채팅)
  │            ├─(스트리머)→ [채널] /channel/:id (방송/클립 탭)
  │            └─(추천)→ 다른 /watch/:id
  ├─(클립)→ [클립] /clips (게임 필터)
  └─(검색)→ [검색] /search?q= (전체/게임/스트리머/클립 탭)
[예외] 없는 스트림/채널→빈상태 / 검색 0건→재검색
```
★ 메인 = 라이브 시청.
