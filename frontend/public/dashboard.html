Advisory 13건은 급하지 않은 개선거리로 기술 부채 목록에 기록 — 실 서버 연동·화면 확장 전 정리.

발견 → 바로 고친 것들

  • 치명
    처음 실행하면 화면이 멈추던 문제
    설정 파일이 공유되지 않아 새로 내려받으면 없는 서버에 연결을 시도. 기본값을 "가짜 데이터로 동작"으로 바꿔 누구나 바로 실행되게 함.
  • 치명
    오류가 나도 안내 화면이 없던 문제
    5개 화면 모두 데이터 불러오기 실패 시 "다시 시도" 버튼이 있는 오류 화면을 보여주도록 연결.
  • 필수
    방송이 하나도 없을 때 화면이 깨지던 문제
    목록이 비어 있어도 "표시할 내용 없음" 화면을 안전하게 표시.
  • 필수
    접근성 4건
    눌리지 않는데 링크처럼 보이던 글자 제거, 흐려서 잘 안 보이던 글자(검색창·편성표·필터)의 색 대비를 진하게 조정.
